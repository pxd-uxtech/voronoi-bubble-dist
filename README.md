# VoronoiBubble

Voronoi treemap visualization engine powering the ○○Bubble series.

Originally created by [@taekie](https://github.com/taekie).  
Copyright (c) 2025 UXtechLab. Licensed under [BUSL-1.1](https://mariadb.com/bsl11/).

## CDN Usage

```html
<!-- Standalone (dependencies bundled) -->
<script src="https://cdn.jsdelivr.net/gh/pxd-uxtech/voronoi-bubble-dist@{hash}/dist/voronoi-treemap.standalone.umd.js"></script>

<!-- Minified (peer deps required: d3, d3-voronoi-treemap, etc.) -->
<script src="https://cdn.jsdelivr.net/gh/pxd-uxtech/voronoi-bubble-dist@{hash}/dist/voronoi-treemap.min.js"></script>
```

## Files

| File | Description |
|------|-------------|
| `voronoi-treemap.standalone.umd.js` | All deps bundled, UMD (browser `<script>`) |
| `voronoi-treemap.standalone.js` | All deps bundled, ESM |
| `voronoi-treemap.umd.js` | UMD, peer deps required |
| `voronoi-treemap.esm.js` | ESM, peer deps required |
| `voronoi-treemap.min.js` | Minified UMD, peer deps required |
